## 💡 변경 기록 방법

PM이 config 파일 수정 후:
```
"프로젝트 목표가 바뀌었어. 기록해줘"
"@park이 합류했어. 백엔드 담당이야"
"마일스톤이 1주 연기됐어"
```

AI-PM이 자동으로:
1. 이 changelog에 기록
2. 필요시 roadmap 재조정 제안
3. 영향받는 팀원에게 알림 제안
